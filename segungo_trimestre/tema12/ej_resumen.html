Resumen de XPath

XPath es un lenguaje que permite acceder y buscar información dentro de documentos XML.

¿Qué es XPath?

XPath es un lenguaje que se utiliza para navegar por un documento XML y seleccionar nodos. Funciona como una ruta (path), similar a las carpetas de un ordenador.

¿Para qué sirve?

  • Acceder a nodos de XML
  • Extraer datos
  • Visualizar elementos
  • Hacer cálculos

Tipos de nodos

  • Raíz ( / )
  • Elementos
  • Atributos
  • Texto

Sintaxis básica

/concesionario/coche → selecciona los coches

//coche → selecciona todos los coches

//coche[2] → segundo coche

//coche/@matricula → atributo matrícula

Predicados

Se usan para filtrar resultados:

//coche[precio > 20000]

//coche[color="Azul"]

Funciones

  • text() → obtiene el texto
  • last() → último elemento
  • count() → contar elementos

Resumen

XPath es muy útil para trabajar con XML porque permite buscar información de forma rápida. Se usa mucho junto con XSLT y otras tecnologías.
