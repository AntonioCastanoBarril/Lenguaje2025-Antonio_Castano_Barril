Resumen de XML

XML es un lenguaje que permite almacenar y transportar datos de forma estructurada.

¿Qué es XML?

XML (eXtensible Markup Language) es un lenguaje de marcado que se utiliza para definir, almacenar y transportar datos de manera estructurada.

Características principales

  • Es extensible (puedes crear tus propias etiquetas)
  • Es legible tanto por humanos como por máquinas
  • Permite organizar la información en forma jerárquica
  • Se usa para intercambio de datos entre sistemas

Estructura de un XML

  • Elemento raíz
  • Elementos hijos
  • Atributos
  • Texto (contenido)

DTD y Esquemas

Se utilizan para definir la estructura que debe tener un documento XML y validar que esté bien formado.

Usos de XML

  • Intercambio de datos
  • Configuración de aplicaciones
  • Servicios web
